# Brand Images

This directory contains Good Roots Network (GRN) brand images.

## Logo Files (To be provided by user)

Place the following logo files in this directory:

- **logo.svg**: Stacked logo with icon and text (primary logo)
- **logo-horizontal.svg**: Logo with text beside icon (for headers)
- **logo-icon.svg**: Icon only for compact spaces (mobile header)
- **social-share.png**: 1200x630px image for Open Graph social sharing

## Logo Guidelines

- Use SVG format for logos to ensure scalability
- Optimize SVG files using SVGO or similar tools
- Ensure logos work on both light and dark backgrounds
- Maintain aspect ratio across all variants
- Keep file sizes small for optimal performance
